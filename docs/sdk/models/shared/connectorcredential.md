# ConnectorCredential

ConnectorCredential is used by a connector to authenticate with conductor one.

## Example Usage

```typescript
import { ConnectorCredential } from "conductorone-sdk-typescript/sdk/models/shared";

let value: ConnectorCredential = {};
```
