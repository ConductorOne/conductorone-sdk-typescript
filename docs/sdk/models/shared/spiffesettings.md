# SPIFFESettings

SPIFFESettings is the kind-specific configuration block for SPIFFE
 trust-domain providers (issuer_url = spiffe://<trust-domain>).

## Example Usage

```typescript
import { SPIFFESettings } from "conductorone-sdk-typescript/sdk/models/shared";

let value: SPIFFESettings = {};
```
