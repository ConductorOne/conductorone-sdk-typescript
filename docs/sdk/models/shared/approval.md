# Approval

The Approval message.

This message contains a oneof named typ. Only a single field of the following list may be set at a time:
  - users
  - manager
  - appOwners
  - group
  - self
  - entitlementOwners
  - expression
  - webhook
  - resourceOwners


## Example Usage

```typescript
import { Approval } from "conductorone-sdk-typescript/sdk/models/shared";

let value: Approval = {};
```
