# TFFieldMapping

TFFieldMapping describes how one field of a C1 API object maps to one
 attribute of a Terraform block.

## Example Usage

```typescript
import { TFFieldMapping } from "conductorone-sdk-typescript/sdk/models/shared";

let value: TFFieldMapping = {};
```
