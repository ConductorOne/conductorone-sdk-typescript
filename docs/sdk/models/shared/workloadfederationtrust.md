# WorkloadFederationTrust

WorkloadFederationTrust represents a per-SP trust policy that references
 a tenant-level provider and defines a CEL condition for claim matching.

## Example Usage

```typescript
import { WorkloadFederationTrust } from "conductorone-sdk-typescript/sdk/models/shared";

let value: WorkloadFederationTrust = {};
```
