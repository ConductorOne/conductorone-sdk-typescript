# PaperSecret

PaperSecret is the API view of a secret (combines Vault + PaperVault fields).
 The vault_id is the primary identifier (Vault.id).

## Example Usage

```typescript
import { PaperSecret } from "conductorone-sdk-typescript/sdk/models/shared";

let value: PaperSecret = {};
```
