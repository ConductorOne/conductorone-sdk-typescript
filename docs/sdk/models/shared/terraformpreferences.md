# TerraformPreferences

TerraformPreferences groups the user's preferences for the "Show
 Terraform code" feature.

## Example Usage

```typescript
import { TerraformPreferences } from "conductorone-sdk-typescript/sdk/models/shared";

let value: TerraformPreferences = {};
```
