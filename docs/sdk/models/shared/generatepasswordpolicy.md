# GeneratePasswordPolicy

GeneratePasswordPolicy defines inline password generation rules.

This message contains a oneof named character_rules. Only a single field of the following list may be set at a time:
  - noRestrictions
  - customCharacters
  - excludedCharacters

```typescript
import { GeneratePasswordPolicy } from "conductorone-sdk-typescript/sdk/models/shared";

let value: GeneratePasswordPolicy = {};
```
