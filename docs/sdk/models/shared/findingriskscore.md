# FindingRiskScore

FindingRiskScore is a composite risk score with decomposable factors.
 Three-score model: score (effective), system_score (latest calculation),
 original_score (immutable from first detection).

## Example Usage

```typescript
import { FindingRiskScore } from "conductorone-sdk-typescript/sdk/models/shared";

let value: FindingRiskScore = {};
```
