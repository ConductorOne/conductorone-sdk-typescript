# AutomationTrigger

Automation Triggers

This message contains a oneof named kind. Only a single field of the following list may be set at a time:
  - manual
  - userProfileChange
  - appUserCreated
  - appUserUpdated
  - usageBasedRevocation
  - userCreated

import { AutomationTrigger } from "conductorone-sdk-typescript/sdk/models/shared";
