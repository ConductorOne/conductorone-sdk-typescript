# AccessReview

The AccessReview message.

This message contains a oneof named setup_metadata. Only a single field of the following list may be set at a time:
  - singleApp
  - multiApp
  - bindings


## Example Usage

```typescript
import { AccessReview } from "conductorone-sdk-typescript/sdk/models/shared";

let value: AccessReview = {};
```
